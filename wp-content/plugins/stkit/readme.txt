== Description ==

Extended functionality for WordPress themes.

This plugin allows to:

<ul>
<li>Install a custom logo & favicon</li>
<li>Install the Google Analytics tracking code</li>
<li>Put your own copyrights on footer</li>
<li>Write a posts with different formats</li>
<li>Get an additional blog templates besides default</li>
<li>Tweak a post page within few clicks</li>
<li>Add a sitewide data\info on post page</li>
<li>Add subtitle for posts, page and projects</li>
<li>Enable a nice time and post views</li>
<li>Add an unlimited number of custom sidebars</li>
<li>Set an unique sidebar for each single page, post or project</li>
<li>Change a position of sidebar or disable it on each single page, post or project</li>
<li>Enable breadcrumbs</li>
<li>Enable <a href="http://strictthemes.com/documentation/widgets/">custom widgets</a></li>
<li>Enable a bunch of useful <a href="http://strictthemes.com/documentation/shortcodes/">shortcodes</a></li>
<li>Insert shortcodes quickly</li>
<li>Enable portfolio</li>
<li>Enable all available portfolio templates</li>
<li>Change a scheme and number of footer sidebars</li>
<li>Enable social icons</li>
<li>Install an any of 600+ custom fonts within few clicks</li>
<li>Tweak a color scheme and make an unique styles with easy</li>
<li>Add a custom CSS without bloating the header of HTML document</li>
<li>Import a theme settings from demo-site</li>
<li>Disable admin bar on the front-end</li>
<li>Disable responsive layout</li>
<li>Remove developer's copyrights from footer</li>
<li>and more...</li>
</ul>
